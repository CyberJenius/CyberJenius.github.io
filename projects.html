This Website!

This project allowed for me to get more experience with web development and design. This project was mostly CSS and HTML with a very small amount of javascript for effects. In general, this project was a lot of fun for me to do. Most of my learning for this project surrounded CSS however, I learned a lot more about HTML elements and features as well. I had a lot of fun changing CSS and HTML properties and visually seeing the changes. Working to fit a design in my head was difficult but it was extremely rewarding whenever I would get a page designed, programmed, and working in a way I liked. I also created a github account and this is my first project in which I used git for version control. This was an extremely interesting expereince and going forward, I will be using github for all of my programming projects. It was fun to get a better grasp on version control, how it works, what it really is, and when to commit code changes. As someone who loves documentation, I greatly see the usefulness of this tool for comparisons, updates, and changes to any code.
